Lights Out Game - Addendum

Process and Approach

The development of the Lights Out game involved a structured process to ensure functionality, user interactivity, and visual appeal. Below is a detailed account of the steps and decisions made:

  • Planning the Game Logic: The core logic for the Lights Out game was planned first, focusing on toggling lights and setting a win condition.
  • User Interface Design: The grid layout and intuitive button controls were implemented for ease of use.
  • Styling: CSS enhancements were applied to make the game visually appealing and responsive.
  • Dynamic Updates: Real-time updates for stats like Target, Moves, and Time were implemented for a better user experience.

Challenges Faced

  • Ensuring Random Yet Solvable Configurations: This was addressed by simulating random clicks on the board to ensure solvability.
  • Tracking Moves and Time: Clear logic was used to reset and track these values dynamically during gameplay.
  • Responsive Design: Flexbox and responsive styling techniques were used to make the game layout adapt to different screen sizes.
  • Integrating Additional Features: The Addendum button was aligned beside the New Game button using Flexbox, ensuring consistent layout and functionality.

Creative Choices

  • UI Enhancements: Clear visual distinction between toggled and non-toggled cells using contrasting colors.
  • Dynamic Game Info: A random Target value was generated at the start of each game for added variety.
  • Flexibility in Addendum: The Addendum button was designed as a placeholder for future enhancements or instructional content.

Insights Gained

Through this project, several key insights were gained:

  • Breaking the logic into reusable functions improved code modularity and maintainability.
  • Subtle animations and polished visuals greatly enhance user engagement.
  • Responsive design is crucial for creating user-friendly layouts across devices.
  • Iterative testing is essential to identify edge cases and ensure smooth functionality.

Conclusion

The Lights Out game development process was both challenging and rewarding. By focusing on clear logic, interactive design, and user experience, a functional and visually appealing game was created. The flexibility to add features like the Addendum button demonstrates the scalability of the project, which can be extended in future iterations to include new challenges, themes, or additional levels.
